Launched using node.js.
Before launching, database must be created and tables must be inserted according to the project schemas.

app.js contains routing and execution of sql queries,
packaje.json lists the details such as dependencies and entry point of the project,
node modules folder contains the required files for the dependencies,
views folder contains the .ejs files which make up the front end output of the project,
views/partials folder contains the header and footer partial included in all other ejs files.
